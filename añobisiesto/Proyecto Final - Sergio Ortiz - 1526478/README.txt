El proyecto esta desarrollado en Python usando libreria PyGame. La libreria es open source y puede ser descargada desde su pagina oficial http://www.pygame.org/ para la version de python que se tenga instalada.

Pasos para ejecutar el programa.
1) Situese en la carpeta ZeldaCode desde una terminal, ahí encontrará todo el codigo fuente del juego

2) El programa principal es el archivo de nombre: main.py

3) Para ejecutar el juego:

$->: python main.py
